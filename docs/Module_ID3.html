Module ID3

This module models provides methods, constants and classes to handle ID3 version 1.x and version 2.x tags for audio files.


Provided Functionality:

  • Reads and parses audio files for ID3-tags
  • Writes audio files and attaches ID3-tags to them    # not yet implemented
  • Provides both high-level and low-level classes for handling files with ID3-tags


Module Constants:

  • ID3::SUPPORTED_SYMBOLS[version]

Module Methods:

  • ID3.hasID3v1tag?(filename)
  • ID3.hasID3v2tag?(filename)
  • ID3.hasID3tag?(filename)
  • ID3.removeID3v1tag(filename)              # ID3v1 tags aren't really very useful - that's why there is a special method to remove them

Classes:

  • ID3::AudioFile
  • ID3::Tag1
  • ID3::Tag2
  • ID3::Frame

Examples:

	> require 'id3'
	=> true

	> ID3.hasID3tag?("mp3/a.mp3")
	=> "1.0 2.3.0"

	> ID3.hasID3tag?("mp3/b.mp3")
	=> "2.2.0"

	> ID3.hasID3tag?("mp3/c.mp3")
	=> "1.1"

	> ID3.hasID3tag?("mp3/d.mp3")
	=> "1.1 2.3.0"

	> `cp mp3/d.mp3 mp3/delete.mp3`
	=> true

	>  ID3.hasID3tag?("mp3/delete.mp3")
	=> "1.1 2.3.0"
	>  ID3.removeID3v1tag("mp3/delete.mp3")
	=> true
	>   ID3.hasID3tag?("mp3/delete.mp3")
	=> "2.3.0"

	> version = ID3.hasID3v2tag?("mp3/b.mp3")
	=> "2.3.0"
