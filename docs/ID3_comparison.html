Comparison of ID3 Standards
by Tilo Sloboda <tools@unixgods.org>


This document contains a side-by-side comparison over the frames used in the 3 different versions of the informal ID3v2 standard: v2.2.0, v2.3.0 and v2.4.0 and the ID3 versions 1.0 and 1.1

This document is a compilation of information which is available in the documents listed below. The original documents Copyright notices apply.


    ID3 v1.0 :     http://www.id3.org/id3v1.html
    ID3 v1.1 :     http://www.egr.uri.edu/~kovacsp/mp3/ID3-Tag.html


    ID3 v2 2.0:     http://www.id3.org/id3v2-00.txt
    ID3 v2 3.0:     http://www.id3.org/id3v2.3.0.txt
    ID3 v2 4.0:     http://www.id3.org/id3v2.4.0-changes.txt
                        http://www.id3.org/id3v2.4.0-structure.txt
                        http://www.id3.org/id3v2.4.0-frames.txt

Please refer to the original documents listed above, for any information
regarding the ID3v2 standards.

References:
    id3lib-3.8.0   id3lib-3.8.0/include/id3/globals.h

                   and LEADARTIST was shortened to ARTIST.
